
# Lotto Number Generator

## Overview

This is a simple web application that generates random lotto numbers.

## Features

- Generate 6 unique random numbers between 1 and 45.
- Display the generated numbers in a user-friendly interface.
- The numbers are color-coded based on their range.

## Project Structure

- `index.html`: The main HTML file.
- `style.css`: The CSS file for styling.
- `main.js`: The JavaScript file for functionality.
